Система "Климатические риски"

Инструкция

  • Для открытия базы — воспользуйтесь меню "Файл" → "Открыть базу" (или сочетанием клавиш Ctrl+o).
  • Для добавления данных — воспользуйтесь меню "Данные" → "Добавление данных об ОЯ"
    • из загруженной таблицы Excel;
    • автоматически с сайта ВНИИГМИ.
  • Для расчёта значений климатических рисков — воспользуйтесь меню "Расчёт рисков"
  • Для исследования методов нормирования данных — воспользуйтесь меню "Анализ" → "Графики для нормирования"
